Quick start

  1. Open this example spreadsheet.
  2. Copy to your Google Drive by selecting "File" > "Make a copy...".
  3. Fill in your own data.
  4. Create your visualization! Give it a name, then paste in its URL.

Overview

Network Mapper uses a Google Spreadsheet as a datastore. This allows you to customize your data set, its associated meta-data and the much of the visualisation's appearance. The spreadsheet has 3 tabs: NODES, CATEGORIES and STYLES.

NODES is where data points and associated meta data is defined. CATEGORIES is a list of category names that you want to associate with your nodes, as well as a style class name for each category node and label. STYLES is a custom stylesheet where you can customize the appearance of your nodes and labels.

Data types

List of supported data types and how they should be used.

Column Description
NAME The name of your node. A node is a single data point with some associated meta data.
CATEGORIES One or more categories for your node. Nodes are connected by category. Categories may be defined in the CATEGORIES tab of the spreadsheet where they can be given a style class which determines their visual appearance. See Customizing styles section below.
IMPORTANCE The relative importance of your node. A larger value will result in a larger node radius.
SHORT DESCRIPTION An introductory paragraph or abstract that will appear in the popup when clicking on a node.
LONG DESCRIPTION The main body of text that will appear in the popup when clicking on a node.
CONTEXT URL A link to an external website that will appear below the long description in the popup when clicking a node. Note: Valid YouTube URLs will automatically be rendered into a YouTube video player.
CREDIT The name of a person or source for the current node data.
NODE-STYLE A class name that provides styling information for your node. See Customizing styles section below.
LABEL-STYLE Note: there is a special class called always-shown-label which allows you to control whether or not node labels are automatically hidden. This can be provided in addition to other classes as a mixin e.g. your-custom-class always-shown-label will render the node label with your own class but make sure the label is not automatically hidden.

Customizing styles

Column Description
CLASSNAME A name for your style class. This can be entered into any of the NODE-STYLE or LABEL-STYLE columns to give your data point a custom appearance.
STYLES A list of space-delimited style attributes. Network Mapper uses D3.js and supports SVG style properties. Some examples are provided in the sample spreadsheet.
